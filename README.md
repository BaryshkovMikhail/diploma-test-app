# 🎓 Diploma Test Application

Простое тестовое приложение на базе nginx для демонстрации развёртывания в Kubernetes кластере в рамках дипломного проекта.

## 📋 Описание

Приложение представляет собой статический веб-сайт, размещённый в nginx-контейнере. Предназначено для:
- Демонстрации работы CI/CD пайплайна
- Тестирования развёртывания в Kubernetes
- Проверки мониторинга и логирования

## 🚀 Быстрый старт

### Локальная сборка и запуск

```bash
# Сборка образа
docker build -t diploma-test-app:latest .

# Запуск контейнера
docker run -d -p 8080:80 --name test-app diploma-test-app:latest

# Проверка
curl http://localhost:8080
```
